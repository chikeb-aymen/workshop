# Problem: Payment Processing System

###### (👀 don't cheat if you want to learn), take your time to learn.

**Your company is building a payment platform.**

**It must support:**

- Credit Card payments
- PayPal payments
- Crypto payments

**Requirements:**

- New payment methods should be added easily
- Existing code should not be modified
- Avoid long conditional logic

Your task is to design a flexible architecture.
