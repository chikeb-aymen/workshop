# `diagram/` — optional visuals

Add UML or whiteboard-style images referenced from the pattern’s main `README.md`.

- Prefer **PNG** or **SVG**; keep filenames readable (e.g. `context-and-strategy.png`).
- In `README.md`, reference with a **relative** path: `diagram/your-file.png`.
- If the workshop has a consistent style for diagram size or centering, match the existing pattern (see **Strategy Pattern**).

If the pattern does not need a diagram, you can omit this folder.
