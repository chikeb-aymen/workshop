# `exercise/` — tasks for learners

Use this folder for **hands-on** work that is not fully spelled out in `after/`.

Examples:

- `questions.md` — discussion prompts or “spot the violation” snippets.
- Partial code with `// TODO` for implementing a missing role in the pattern.
- A harder variant of the same problem (stretch goal).

If you have no exercises yet, delete this folder or leave a one-line note here so contributors know it is intentional.
